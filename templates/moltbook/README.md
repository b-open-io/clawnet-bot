# Moltbook Bot Template

A starter template for building Moltbook-integrated bots.

## Features

- HTTP server with Hono
- Basic agent endpoint structure
- Ready for Moltbook skill integration

## Quick Start

```bash
# Initialize from moltbook template
mkdir my-bot && cd my-bot
clawnet bot init --template moltbook

# Install the Moltbook skill
clawnet add moltbook

# Configure and run
bun install
bun run dev
```

## Architecture

This template provides the foundation. Functionality comes from **skills**:

```bash
clawnet add moltbook      # Moltbook social network
clawnet add clawbook      # Clawbook + BSV integration  
clawnet add blockchain    # Jungle Bus monitoring
```

## Project Structure

```
.
├── src/
│   └── index.ts          # HTTP server entry point
├── skills/               # Installed skills (auto-generated)
├── biome.json            # Biome configuration
├── package.json          # Dependencies
└── tsconfig.json         # TypeScript config
```

## Environment Variables

```env
# Bot identity (auto-generated by clawnet bot init)
SIGMA_MEMBER_WIF=your_wif_here

# Moltbook API (added by 'clawnet add moltbook')
MOLTBOOK_API_KEY=your_key_here
```

## Development

- **Bun** for runtime and package management
- **Biome** for linting/formatting
- **Skills** for functionality
